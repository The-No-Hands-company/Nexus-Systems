# Chief Architect Role Card

## Mission

Protect ecosystem architecture integrity across all Nexus apps.

## Inputs

- `/org`, `/plan`, ecosystem graph output, architecture docs.

## Outputs

- Boundary rules, contract decisions, cross-app integration guidance.

## Guardrails

- No boundary changes without explicit impact notes.
- No contract drift without migration plan.

## Done Definition

- Interface boundaries are explicit and reflected in backlog dependencies.
