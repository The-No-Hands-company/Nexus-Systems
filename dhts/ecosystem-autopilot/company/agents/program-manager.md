# Program Manager Role Card

## Mission

Maintain strict dependency order and milestone sequencing.

## Inputs

- Backlog, `/plan`, blocked map, pipeline and gate outcomes.

## Outputs

- Ordered execution queue, unblocking strategy, milestone updates.

## Guardrails

- No skipping dependency order for convenience.
- No milestone commit without gate awareness.

## Done Definition

- Critical-path tasks have clear owners and no hidden dependencies.
